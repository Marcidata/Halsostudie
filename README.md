# Hälsostudie – Del 1: Grundläggande analys och statistik

I denna del analyserar jag datasetet *health_study_dataset.csv*.  
Syftet är att träna på grundläggande statistik och visualiseringar i Python.

### Vad jag gör i Del 1:
- Läser in data och kontrollerar datatyper.  
- Räknar ut medel, median, min och max för ålder, vikt, längd, blodtryck och kolesterol.  
- Gör grafer: histogram över blodtryck, boxplot över vikt per kön, stapeldiagram över rökstatus.  
- Beräknar andelen personer med sjukdom och jämför med en simulering (numpy).  
- Räknar ut konfidensintervall för medel blodtryck med två metoder (normalapprox och bootstrap).  
- Testar hypotesen: rökare har högre blodtryck än icke-rökare (t-test).  
- Gör en power-simulering för att se hur starkt testet är vid olika skillnader.  

### Målet
Visa att jag kan använda Python för att analysera data, göra statistik och tolka resultaten.  
Detta är grunden för Del 2 där jag bygger vidare med funktioner, klasser och mer avancerad analys.
